# WebSkripsi 🦉
Sistem Informasi Satwa Dilindungi di Jawa Timur

## 📖 Deskripsi
WebSkripsi adalah aplikasi berbasis web yang dikembangkan menggunakan **Flask** dan **Folium** untuk menampilkan peta interaktif persebaran satwa dilindungi di Jawa Timur. Sistem ini dirancang sebagai media edukasi dan pelaporan konservasi, dengan fitur admin untuk pengelolaan data satwa dan lokasi konservasi.

## ⚙️ Teknologi yang Digunakan
- **Flask** – Framework utama untuk pengembangan web.
- **SQLAlchemy** – ORM untuk pengelolaan database PostgreSQL.
- **Flask-Migrate** – Migrasi database otomatis.
- **Folium** – Visualisasi peta interaktif.
- **Python-dotenv** – Pengelolaan variabel lingkungan dari file `.env`.

## 🧩 Fitur Utama
- Peta interaktif dengan filter kombinasi (Nama Satwa, Lokasi Konservasi, Jenis Satwa).
- CRUD data satwa dan lokasi konservasi untuk admin.
- Tampilan detail satwa dengan gambar dan deskripsi habitat.
- Sistem login admin dengan keamanan berbasis session.
- Integrasi database PostgreSQL.
